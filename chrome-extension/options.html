HttpsTypeSearch Options

Configure the local API endpoint, search behavior, and URL matching rules used by the extension. If Chrome reports Failed to fetch for a local HTTPS endpoint, the extension cannot bypass certificate trust checks — use a trusted certificate or switch to local HTTP.

Local API connection

Only localhost and 127.0.0.1 endpoints are allowed by the extension manifest.

Use a local endpoint such as https://127.0.0.1:19456 or http://localhost:19456.

Leave empty only if your local API does not require a token.

Search and autofill

These toggles keep the current functionality while making the behavior easier to understand.

URL term and key matching

Search term comes from current tab URL. Results are shown only when matching configured keys.

Only hostname or hostname:port is used for term=. Path/query/hash are ignored.
One keyword per line, or use commas. Fuzzy match is case-insensitive and checks whether a CustomFields key contains the keyword.
